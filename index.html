The Ivy Leaf Project

Through research-based reports on urban architecture, engineering, and environmental innovation, the Ivy Leaf Project is sharing technologies and ideas that are shaping a more sustainable future.

The Ivy Leaf Project is a calm home for reports, case studies, and hopeful ideas about sustainable architecture, green engineering, and climate-smart design. Visitors can read the archive, while only the owner can add new stories.
